Exploratory Data Analysis of Global Air Quality Data¶

Problem Statement¶

Air pollution poses serious risks to human health and the environment. This project uses global air quality data to explore patterns
,trends, and correlations among pollutants across countries and time, aiming to identify pollution hotspots and key insights.

Objective¶

Briefly explain what air quality is and why it matters.
To analyze global air quality patterns, identify trends, and explore correlations among pollutants, time, and location.

Scope of Analysis¶

Univariate: Distribution of pollutants (PM2.5, PM10, NO2, SO2, O3, CO).
Bivariate/Multivariate: Relationships between pollutants, weather, and geography.
Time-Series: Trends over time (daily, monthly, yearly).
Geospatial: Regional comparison of pollution levels across countries/cities.

# Checking the AIQ levels ,interpreting it by analysing is it good,bad,moderate for health
def aqi_category(aqi):
    if aqi <= 50:
        return "Good"
    elif aqi <= 100:
        return "Moderate"
    elif aqi <= 150:
        return "Unhealthy for Sensitive Groups"
    elif aqi <= 200:
        return "Unhealthy"
    elif aqi <= 300:
        return "Very Unhealthy"
    else:
        return "Hazardous"

data['AQI_Category'] = data['AQI'].apply(aqi_category)
